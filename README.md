# COVID-19 Tracker App
Android application for tracking the COVID-19 outbreak around the world with [coronavirus-tracker](https://github.com/ExpDev07/coronavirus-tracker-api) api (work in progress)

### Using
* RxAndroid
* Retrofit
* Android Maps Utils
* MPAndroidChart
* ThreeTenABP
